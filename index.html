Pokernow Log Analyzer

Python script for calculating statistics from Pokernow game logs. Over the pandemic, the Pokernow site has surged in popularity and unlike established sites like PokerStars, there is no software that accomodates its logs. When I finish implementing the calculations I plan on turning it into a webapp with the Python streamlit library.
